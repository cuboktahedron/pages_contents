IkaRecord(イカレコード)

このツールでは任天堂様のゲーム、Splatoon（スプラトゥーン）の
戦績の管理が行えます。

こんな感じで登録して、

こんな感じに結果が出せます。

動作環境

Windows7(32bit/64bit)
.Net Framework4.5.2での動作を確認しています。

ダウンロード

1.0.0
リンク先の画面上部のアイコンからダウンロードしてください。

マスタファイル
右クリック ＞ 対象をファイルに保存 でダウンロードしてください。
